# AddressBook

## Project Description
The AddressBook application is a simple contact management system. It uses the file system (pretending to be a database) where contact details are stored and managed in CSV files. The app enables users to create, read, update, and delete (CRUD) contact information effectively.

## Features
- Add new contacts with ease.
- Modify existing contact information.
- View all stored contacts.
- Delete unnecessary contacts.
- Lightweight, fast, and file-based storage management.

## Technologies Used
- Java
- CSV file handling

## Setup Instructions
Follow these steps to build and run the AddressBook application:

1. **Prerequisite:** Ensure Java (version 8 or above) is installed on your machine.

2. Clone the repository:
```bash
git clone https://github.com/avinash25mis/AddressBook.git
cd AddressBook
```

3. Compile the application:
```bash
javac -d bin src/com/addressbook/Main.java
```

4. Run the application:
```bash
java -classpath bin com.addressbook.Main
```

5. Follow the prompts to manage your address book seamlessly.
